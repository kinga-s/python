a * x + b * y + c = 0

SPECYFIKACJA
Problem: Podanie rozwiązań równania: a * x + b * y + c = 0

DANE WEJŚCIOWE
Trzy liczby a, b i c, będące zmiennymi równania a * x + b * y + c = 0.

DANE WYJŚCIOWE
Rozwiązania równania a * x + b * y + c = 0

LISTA KROKÓW

K01: Jeśli a = 0
K02: 	Jeśli b = 0 Równanie sprowadza się do postaci: c=0
K03: 	    Jeśli c != 0 Wypisz: Równanie nie ma rozwiązań
K04:	    W przeciwnym wypadku: Wypisz: równanie tożsamościowe
K05:	W przeciwnym wypadku: oblicz i wypisz y=-c/b
K06: W przeciwnym wypadku jeśli b = 0: równanie sprowadza się do postaci ax+c=0 => oblicz i wypisz x=-c/a
K07: W przeciwnym wypadku: rozwiązaniem równania jest y=(-c-a*x)/b oblicz i wypisz y=-c/b-(a/b)*x
